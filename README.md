# Skip Animations Mod

This mod lets you skip long uninteractive sequences such as police raid and pollux ceremony.

## How to use

1. Install [BepInEx (Mono build)](https://github.com/BepInEx/BepInEx/releases) into your Hollywood Animal folder (commonly at `/Steam/steamapps/common/Hollywood Animal/`).
2. Launch the game once so BepInEx creates `/Hollywood Animal/BepInEx/plugins/`.
3. Copy `SkipAnimationsMod.dll` into `/Hollywood Animal/BepInEx/plugins/`.
4. Launch the game again.

## Configuration

Config file: `BepInEx/config/hollywoodanimal.skipanimations.cfg`

- `EnableEscSkip` (default `true`)
- `SkipHotkey` (default `Escape`)
